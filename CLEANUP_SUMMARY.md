## Archive Management

The `archive/` folder contains all removed files. You can:

1. **Keep it** - For historical reference
2. **Delete it** - If you don't need old files (saves ~50MB)
3. **Git ignore it** - Add to .gitignore if keeping locally

To delete archive:
```bash
rm -rf archive/  # Linux/Mac
Remove-Item -Recurse -Force archive/  # Windows PowerShell
```
